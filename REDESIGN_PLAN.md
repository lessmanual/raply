# Plan Redesignu Raply: "Editorial Tech"

## 1. Filozofia Wizualna: "Radical Clarity"

Celem jest odejście od generycznego stylu SaaS (ciemny motyw, fioletowe gradienty) na rzecz elegancji magazynu finansowego połączonej z nowoczesnym softwarem.

### Paleta Kolorów
*   **Tło:** "Paper White" (Złamana biel, `#F9F9F7`) - dla redukcji zmęczenia oczu.
*   **Tekst:** "Deep Charcoal" (Grafit, `#1A1A1A`) - zamiast czystej czerni.
*   **Akcent Główny:** "International Orange" (`#FF4F00`) - dla przycisków CTA i kluczowych akcji.
*   **Akcent Finansowy:** "Deep Forest Green" (`#0A4D2E`) - dla pozytywnych wskaźników.
*   **Tryb Ciemny:** "Midnight Navy" (nie czarny).

### Typografia
*   **Nagłówki (Headings):** Szeryfowe (Serif). Celujemy w *Instrument Serif*, *Newsreader* lub *Playfair Display*. Nadaje powagi i charakteru "premium".
*   **Treść (Body):** Czysty Sans-serif (np. *Inter* lub *Satoshi*).
*   **Dane (Data/Numbers):** Monospace (np. *Geist Mono*, *JetBrains Mono*). Podkreśla techniczny aspekt narzędzia.

## 2. Komponenty UI

### Dashboard (Bento Grid 2.0)
*   Układ siatki (Grid).
*   Brak cieni (drop-shadow), zamiast tego subtelne obrysy (border).
*   Sekcja "Pulse": Jedno zdanie podsumowujące stan biznesu (AI Summary) zamiast ściany wykresów na start.

### Raporty ("The Living Document")
*   Stylistyka artykułu z New York Times / Financial Times.
*   Minimalistyczne wykresy (brak zbędnych linii siatki).
*   Komentarze AI jako adnotacje "na marginesie" lub przypięte do punktów danych.

### Nawigacja
*   Odejście od ciężkiego sidebara.
*   Floating Dock (pływający pasek) lub minimalistyczny Top Bar, aby zmaksymalizować przestrzeń roboczą.

## 3. Plan Wdrożenia (Landing Page)

1.  **Setup:** Konfiguracja fontów (Next/Font) i zmiennych CSS (Tailwind).
2.  **Hero Section:**
    *   Abstrakcyjna wizualizacja danych zamiast screenshota.
    *   Duża, odważna typografia szeryfowa.
3.  **Social Proof:** Stylizowane na "cytaty redakcyjne" lub czyste logotypy.
4.  **Features:** Bento Grid prezentujący funkcje w sposób graficzny, nie tylko lista z ikonami.
5.  **CTA:** Minimalistyczne, ale z silnym kolorem akcentowym.
